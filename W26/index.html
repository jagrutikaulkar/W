Tap on each year to view subjects and descriptions.

FE - First Year

  • Engineering Mathematics

    Calculus and algebra basics.

  • Programming in C

    Problem solving and logic building.

  • Engineering Physics

    Fundamental physics concepts.

  • Basic Electrical Engineering

    Circuits and electrical systems.

SE - Second Year

  • Data Structures

    Stacks, queues, trees, graphs.

  • OOP in Java

    Classes, objects, inheritance.

  • DBMS

    SQL and database design.

  • Computer Networks

    Networking fundamentals.

TE - Third Year

  • Web Technologies

    HTML, CSS, JavaScript.

  • Operating Systems

    Process and memory management.

  • Software Engineering

    SDLC and development models.

  • Computer Networks

    Advanced networking concepts.

BE - Final Year

  • Artificial Intelligence

    Machine learning and AI systems.

  • Cloud Computing

    AWS and cloud architecture.

  • Cyber Security

    Security and cryptography basics.

  • Project Work

    Final year implementation project.
