# FAQ

## Why did you use CSS Flexbox/Grid?

Answer: "I used Flexbox for the header layout because it handles alignment perfectly regardless of screen size. I used CSS Grid for the 'Skills' section because it automatically wraps elements into new rows based on the screen width, which makes the design inherently responsive."

## Why use Semantic HTML5 tags?

Answer: "Using tags like <header>, <section>, and <footer> instead of just <div> makes the code more readable for developers and helps search engines understand the structure of my page, which is a best practice for accessibility and SEO."

## How is this page responsive?

Answer: "I implemented a media query in the CSS (@media (max-width: 600px)). This detects when a user is on a mobile device and changes the layout—for example, stacking the header elements vertically—so the content remains easy to read without horizontal scrolling."
